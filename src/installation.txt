OpenCart 4.1.0.1-4.1.0.2 Hungarian language (formal)

A comprehensive Hungarian language pack (formal) for OpenCart 4.1.0.1-4.1.0.2 version! This package includes all necessary language files for the admin interface, frontend, extensions, and the installation process. The translation is available for OpenCart 4.1.0.1-4.1.0.2 versions.

Key Features

- A complete Hungarian language pack (formal) for OpenCart 4.1.0.1-4.1.0.2: backend, frontend, extension, and installation.
- Compatible with OpenCart 4.1.0.1-4.1.0.2 versions.
- The translation is thorough and detailed; we have paid great attention to ensuring that the terminology is consistent across every page, e.g., using "áruház" instead of "üzlet" or "bolt".

Installation Using the Extension Installer

1. Download the "Hungarian language (formal)" package from the OpenCart Marketplace.
2. Log in to the OpenCart 4.1.0.1-4.1.0.2 administration interface.
3. Navigate to the "Extensions" -> "Installer" page.
4. Click on the "Upload" button and select the "ps_hu_formal_language.ocmod.zip" file to upload.
   Note: Ensure that the file name is exactly "ps_hu_formal_language.ocmod.zip", or the installation will fail!
5. You will see the "Hungarian language (formal)" extension in the list, click on the "Install" button.
   Note: The installation may take a while as the system extracts the files; please wait until the message "Success: You have modified extensions!" appears.
6. Navigate to the "Extensions" page and under "Choose the extension type", select "Languages".
7. You will see the "Hungarian language (formal)" extension in the list, click on the "Install" button.
   Note: Applying the language settings may take some time as the system configures the new language; wait until the message "Success: You have modified languages!" appears.
8. Click on the "Edit" button, then enable the language extension and click "Save"; this will activate the language pack.

Caution: If you installed the language pack using the extension installer, then go to the "Languages" section under the "Extensions" page, and click the "Uninstall" button next to the language, it will remove not only the language but also all translations for products, categories, and more!

Manual Installation

1. Download the "Hungarian language (formal)" package "ps_hu_formal_language.zip" from the OpenCart Marketplace.
2. Extract the language package files to a folder.
3. Use your preferred FTP client and navigate to the OpenCart 4.1.0.1-4.1.0.2 installation directory.
4. Upload the following folders: "admin", "catalog", "extension" (and "install" if you are installing OpenCart 4.1.0.1-4.1.0.2 for the first time).
5. Log in to the OpenCart 4.1.0.1-4.1.0.2 administration interface.
6. Navigate to the "System" -> "Localization" -> "Languages" page.
7. Click on the "Add New" button.
8. In the "Language Name" field, enter "Magyar"; in the "Code" field, enter "hu-hu"; in the "Locale" field, enter "hu_HU.UTF-8,hu_HU,hu-hu,hungarian"; finally, enable the language.
9. Click the "Save" button; this completes the installation of the language pack.
